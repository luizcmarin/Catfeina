# Política de Segurança

## Reportando uma Vulnerabilidade

Se você acredita ter encontrado uma vulnerabilidade de segurança no **Catverso**, agradecemos sua ajuda em divulgá-la de forma responsável.

**Por favor, NÃO reporte vulnerabilidades de segurança através de issues públicas no GitHub.**

Em vez disso, envie um e-mail para **luizcmarin@gmail.com** com o assunto "Vulnerabilidade de Segurança no Catverso".

Inclua uma descrição detalhada da vulnerabilidade, os passos para reproduzi-la e, se possível, um exemplo de código ou prova de conceito.

### Nosso Compromisso

- Nós nos comprometemos a investigar todos os relatórios de forma rápida e confidencial.
- Solicitamos que você mantenha a vulnerabilidade confidencial até que tenhamos a oportunidade de lançar uma correção.
- Faremos o nosso melhor para preparar e lançar um patch o mais rápido possível.
- Informaremos publicamente sobre a vulnerabilidade assim que uma correção estiver disponível.

> **Nota:** Como um projeto de código aberto não comercial, atualmente não oferecemos recompensas financeiras (bug bounties).
